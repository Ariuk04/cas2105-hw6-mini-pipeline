# CAS2105 Homework 6 – Mini AI Pipeline

This repository contains code and report for Homework 6 of CAS2105.

## Task
Binary sentiment classification (positive vs. negative) on English sentences.

## Files
- `notebooks/`: Jupyter notebook with data loading, baseline, AI pipeline, and evaluation
- `report/`: Final report PDF

## Method Overview
- Naïve baseline: keyword-based sentiment rules
- AI pipeline: pre-trained DistilBERT sentiment classifier (inference only)

## Results
The AI pipeline significantly outperforms the baseline in both accuracy and F1 score.
See the report PDF for detailed results and qualitative examples.
